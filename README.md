# 렌토 (Lento) - 랜딩 페이지

임신 준비 및 난임 여정을 위한 따뜻한 동반자 서비스

## 기술 스택
- React 18.3.1
- TypeScript
- Tailwind CSS v4
- React Router
- Vite

## 주요 기능
- 📱 반응형 디자인
- 🎨 렌토 브랜드 디자인 시스템
- 🖼️ 인터랙티브 폰 프레임 (드래그/스와이프)
- 📅 사전예약 시스템
- 💬 커뮤니티 (댓글, 좋아요)

## 설치 및 실행

\`\`\`bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
\`\`\`

## 파일 구조
- `/src/app/App.tsx` - 메인 앱
- `/src/app/components/` - 재사용 컴포넌트
- `/src/app/pages/` - 페이지 컴포넌트
- `/src/imports/` - 이미지 assets

## 배포
프로젝트는 Vite로 빌드됩니다.
